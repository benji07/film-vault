# Formatage (Biome)

- Indentation : tabs
- Largeur max : 120 caracteres
- Quotes : doubles (`"`)
- Semicolons : toujours
- Lancer `npm run format` avant de commiter
- Les imports doivent utiliser l'alias `@/*` (jamais de chemins relatifs `../`)
